# BrownRDA
Convert RDA files from Brown University XA30 System to be compatible with Osprey

These scripts were created by Steve C.N. Hui, PhD, George Washington University in 2023 and modified by Heiner N. Raum, PhD, UKM Munster in 2024 (modification: adapttion of geometry header fields of XA30 to VE in io_loadspec_rda_modified)
